## What is Anchor?

Anchor is an offline-first, self-hostable note-taking application that works across web and Android. Notes are stored locally, remain fully editable without an internet connection, and sync automatically across all your devices when you're back online.

## Deploy in minutes

<Steps>
  <Step title="Create a docker-compose.yml">
    ```yaml docker-compose.yml
    services:
      anchor:
        image: ghcr.io/zhfahim/anchor:latest
        container_name: anchor
        restart: unless-stopped
        ports:
          - "3000:3000"
        volumes:
          - anchor_data:/data

    volumes:
      anchor_data:
    ```
  </Step>
  <Step title="Start the container">
    ```bash
    docker compose up -d
    ```
  </Step>
  <Step title="Open Anchor">
    Navigate to [http://localhost:3000](http://localhost:3000) and create your first account.
  </Step>
</Steps>

<Note>
  Anchor includes an embedded PostgreSQL database — no separate database setup is required for most deployments.
</Note>

## Key features

- **Offline editing** — All changes are saved locally and synced automatically when online
- **Rich text formatting** — Bold, italic, underline, headings (H1–H3), ordered and unordered lists, checkboxes
- **Tags** — Create color-coded tags to organize notes by topic or project
- **Note backgrounds** — Customize notes with solid colors and patterns
- **Pin & archive** — Pin important notes to the top; archive notes you want to keep but not see daily
- **Trash & recovery** — Soft-delete notes with a recovery window before permanent deletion
- **Note sharing** — Share individual notes with other users as viewer or editor
- **Search** — Full-text search across note titles and content, locally
- **Admin panel** — Manage users, control registration, and view system statistics
- **OIDC authentication** — Integrate with any OpenID Connect provider
