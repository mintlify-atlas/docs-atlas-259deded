---
title: "Update tag"
description: "Update a tag's name or color"
---

<Info>
  All requests to this endpoint require an `Authorization: Bearer <token>` header.
</Info>

## Request

**PATCH** `/api/tags/:id`

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token. Example: `Bearer eyJhbGci...`
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`.
</ParamField>

### Path parameters

<ParamField path="id" type="string" required>
  The UUID of the tag to update.
</ParamField>

### Body

All body fields are optional. Supply only the fields you want to change.

<ParamField body="name" type="string">
  New display name for the tag. Must be non-empty if provided. Must be unique among this user's tags.
</ParamField>

<ParamField body="color" type="string">
  New color value for the tag (e.g., `#22C55E`). Pass `null` to clear an existing color.
</ParamField>

## Response

Returns the updated tag object.

<ResponseField name="id" type="string" required>
  Unique identifier for the tag (UUID).
</ResponseField>

<ResponseField name="name" type="string" required>
  Updated display name.
</ResponseField>

<ResponseField name="color" type="string">
  Updated color value, or `null` if cleared.
</ResponseField>

<ResponseField name="isDeleted" type="boolean" required>
  Whether the tag is soft-deleted.
</ResponseField>

<ResponseField name="userId" type="string" required>
  ID of the user who owns this tag.
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp of creation.
</ResponseField>

<ResponseField name="updatedAt" type="string" required>
  ISO 8601 timestamp reflecting this update.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl --request PATCH \
  --url https://your-instance.example.com/api/tags/c3d4e5f6-a7b8-9012-cdef-012345678901 \
  --header 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...' \
  --header 'Content-Type: application/json' \
  --data '{
    "color": "#EF4444"
  }'
```

</CodeGroup>
