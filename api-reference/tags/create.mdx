---
title: "Create tag"
description: "Create a new tag"
---

<Info>
  All requests to this endpoint require an `Authorization: Bearer <token>` header.
</Info>

## Request

**POST** `/api/tags`

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token. Example: `Bearer eyJhbGci...`
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`.
</ParamField>

### Body

<ParamField body="name" type="string" required>
  Display name for the tag. Must be non-empty. Tag names are unique per user — attempting to create a tag with a duplicate name returns a conflict error.
</ParamField>

<ParamField body="color" type="string">
  Optional color value to associate with the tag (e.g., a hex string like `#FF5733`). Omit or pass `null` to leave unset.
</ParamField>

<Note>
  Tag names are scoped to the authenticated user. Two different users can have tags with the same name.
</Note>

## Response

Returns the created tag object.

<ResponseField name="id" type="string" required>
  Unique identifier for the tag (UUID).
</ResponseField>

<ResponseField name="name" type="string" required>
  Display name of the tag.
</ResponseField>

<ResponseField name="color" type="string">
  Color value for the tag. `null` if not provided.
</ResponseField>

<ResponseField name="isDeleted" type="boolean" required>
  Always `false` for a newly created tag.
</ResponseField>

<ResponseField name="userId" type="string" required>
  ID of the authenticated user who created the tag.
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp of creation.
</ResponseField>

<ResponseField name="updatedAt" type="string" required>
  ISO 8601 timestamp of last update.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl --request POST \
  --url https://your-instance.example.com/api/tags \
  --header 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "research",
    "color": "#8B5CF6"
  }'
```
