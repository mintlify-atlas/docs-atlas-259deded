---
title: "List shares"
description: "List all shares for a note"
---

<Info>
  All requests to this endpoint require an `Authorization: Bearer <token>` header.
</Info>

## Request

**GET** `/api/notes/:id/shares`

Returns all active and soft-deleted share records for the specified note. Only the note owner can list shares.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token. Example: `Bearer eyJhbGci...`
</ParamField>

### Path parameters

<ParamField path="id" type="string" required>
  The UUID of the note whose shares you want to list.
</ParamField>

## Response

Returns an array of note share objects.

<ResponseField name="[]" type="object[]">
  <Expandable title="NoteShare object" defaultOpen>
    <ResponseField name="id" type="string" required>
      Unique identifier for the share record (UUID).
    </ResponseField>
    <ResponseField name="noteId" type="string" required>
      UUID of the note that was shared.
    </ResponseField>
    <ResponseField name="sharedWithUserId" type="string" required>
      UUID of the user this note is shared with.
    </ResponseField>
    <ResponseField name="permission" type="string" required>
      Access level granted. One of `viewer` or `editor`.
    </ResponseField>
    <ResponseField name="sharedByUserId" type="string" required>
      UUID of the user who created this share.
    </ResponseField>
    <ResponseField name="isDeleted" type="boolean" required>
      Whether this share has been revoked. Revoked shares are retained for sync purposes.
    </ResponseField>
    <ResponseField name="createdAt" type="string" required>
      ISO 8601 timestamp of when the share was created.
    </ResponseField>
    <ResponseField name="updatedAt" type="string" required>
      ISO 8601 timestamp of when the share was last updated.
    </ResponseField>
  </Expandable>
</ResponseField>
